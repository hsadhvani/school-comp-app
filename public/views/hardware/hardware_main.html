Introduction to Hardware
Basic Information

When you start a puzzle, you always start by putting all of your puzzle pieces together on a table. The same applies to the way we build a computer. Every piece of the puzzle needs to be put in the right spot in order to get a computer to run properly.

We discussed in the earlier section that computers required Input and Output. But where does that Input actually go when it gets submitted to a computer. What puzzle pieces make up a computer and what do they actually do? How do they all work together?

A computers puzzle pieces include 3 main categories: I/O, Processing, and Storage.
